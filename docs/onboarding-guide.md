# Onboarding Guide

## Week 1
- Meet your manager and team
- Complete mandatory security training
- Set up development environment using the setup script in /internal/setup
- Read the Employee Handbook

## Week 2
- Shadow team members on current projects
- Get access to required systems (submit request via IT portal)
- Attend department standup meetings

## Key Contacts
- IT Helpdesk: it@company.internal
- HR: hr@company.internal
- Security: security@company.internal

## Tools & Systems
- Slack: internal communications
- Jira: project tracking
- Confluence: documentation
- GitHub: code repositories
